# Lua-wa
Lua-wa (Loo-ah-wa) is a web-application 'framework' for Lua+Nginx.

## Requirements
The following Lua modules are required (I recommend using Luarocks to install):
+ Nginx w/ Lua module
+ Lua 5.1 / LuaJIT 2
+ Lua cjson
